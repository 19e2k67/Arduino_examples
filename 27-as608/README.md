# Arduino UNO R3_AS608_OLED
Based on Arduino and AS608, add, delete, and verify fingerprints, and display results on OLED screens

AS608-->Arduino UNO  
电源要接3.3V，5V可能会烧坏  
GND-->GND  
TX-->2  
RX-->3(这俩引脚可以修改成自己想要的)  


OLED-->Arduino UNO  
GND-->GND  
VCC-->VCC  
SCL-->SCL  
SDA-->SDA  


KEY1 -->4  
KEY2 -->12  
KEY3 -->6  
KEY4 -->8  
KEY5 -->9
